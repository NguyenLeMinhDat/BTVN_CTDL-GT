## **Giới thiệu**
Chương trình cho người dùng thấy được cách sắp xếp của một vài loại thuật toán phổ biến:
- Insertion Sort
- Selection Sort
- Bubble Sort
- Merge Sort
- Quick Sort

## **Hướng dẫn cách chạy**
Giả dụ rằng bạn đã cài đặt trình biên dịch **gcc** hoặc bất kì trình biên dịch C nào khác. Hãy mở terminal lên, đảm bảo rằng bạn đang ở đúng vị trí thư mục, chạy câu lệnh sau để chạy chương trình:  
```
chmod +x ./run.sh
./run.sh
```
